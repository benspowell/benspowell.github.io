Congo Online Store
Databases Class - Fall 2018

This project was made as a part of a group assignment for a Database Systems class. It's essentially a mock web store control panel, with a order form shop section.

The main emphasis for this project was on planning, constructing and implementing a database from scratch. MySql was used for the database management, and the site uses php to retrieve and alter data from the database.

As a long(ish) term project, it allowed me to become familiar with the process of planning, implementing, and delivering a project based on limited requirements.

Advanced Energy Vehicle
Fundamentals of Engineering Class - Fall 2016

A semester-long engineering project completed by every first-year engineering student at OSU. The purpose of the project is to create a vehicle which runs on a suspended metal rail, hooking up a cart and carrying it to designated stops on the rail.

An arduino controller was used alongside physical components for the frame, propeller motors, servo motor, etc. The project included several extensive technical reports aimed to imitate large-scale engineering projects. MATLAB was used to analyze data taken from various tests of the vehicle.

The project was successfully completed and included a final test, a poster, a video, a presentation, and several reports and coding projects, all of which can be viewed on the project website.
